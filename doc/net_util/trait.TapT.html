fn new(vnet_hdr: bool, multi_vq: bool) -> Result<Self>[src]

Create a new tap interface. Set the vnet_hdr flag to true to allow offloading on this tap, which will add an extra 12 byte virtio net header to incoming frames. Offloading cannot be used if vnet_hdr is false. set 'multi_vq' to ture, if tap have multi virt queue pairs

fn into_mq_taps(self, vq_pairs: u16) -> Result<Vec<Self>>[src]

Change the origin tap into multiqueue taps, this means create other taps based on the origin tap.
